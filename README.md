# Game - Resgate :helicopter:

###Descrição: :helicopter:

Jogo de resgate, criado utilizando HTML5, CSS3, JavaScript e JQuery.<br> Projeto realizado durante <strong> Bootcamp Take Blip Web Developer - DIO com mentoria de Denilson Bonatti</strong>.

###Como jogar? :helicopter:

Utilize as teclas:
 <li><strong>W</strong> = Movimenta o helicóptero para cima;</li>
  <li><strong>S</strong> = Movimenta o helicóptero para baixo/resgata o refém;</li>
    <li><strong>D</strong> = Movimenta o helicoptero para a direita/atira.</li>
